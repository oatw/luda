Cross-Framework

Build reusable components with Luda, then package them in any framework with less effort, no matter in a front-end framework or a back-end framework.

Component-Based

Focus on component functions under a straightforward and familiar feeling like using jQuery, without worrying about lifecycle handling and cache management.

Built-In Components

Use the themeable built-in components which are designed under a progressive architecture, save your time on building commonly used components.
